## What Is It?

The **OCPP Simulator** is a visual, zero-install charge point emulator that runs entirely in the browser. It uses the `BrowserOCPPClient` from `ocpp-ws-io/browser` to establish real WebSocket connections to a Central System (CSMS) and emulate the full OCPP message lifecycle тАФ from `BootNotification` all the way through `StartTransaction`, `MeterValues`, and `StopTransaction`.

It supports **OCPP 1.6, 2.0.1, and 2.1** and is designed for:

- **Development** тАФ test your CSMS handlers without real hardware
- **Integration testing** тАФ reproduce edge cases and race conditions
- **Protocol exploration** тАФ inspect raw OCPP frames in real time

| UI        | Next.js + React        | Renders panels, controls, logs      |
